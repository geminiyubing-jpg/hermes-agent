🔄 数据流与请求生命周期

一条消息从用户发送到回复的完整处理流程

1
消息接收 — 用户通过平台发送消息 → 平台适配器(如飞书/Discord/Telegram)接收原始消息
↓
2
消息解析与去重 — 解析消息格式(文本/图片/文件/按钮) → 消息去重 → 提取元数据(平台/用户/会话)
↓
3
会话加载与上下文构建 — SessionDB 加载历史 → PromptBuilder 构建系统提示 → 注入记忆/技能/策略/平台提示
↓
4
LLM 推理与工具调用 — 发送到模型API → 流式响应解析 → 检测工具调用 → 并行执行工具 → 返回结果 → 继续推理循环
↓
5
流式输出与适配 — StreamConsumer 处理流式文本 → 按平台消息长度分片 → 格式适配(Markdown/HTML/纯文本)
↓
6
消息发送与持久化 — 通过平台适配器发送回复 → 会话存入SessionDB → 更新使用统计 → 触发自演进遥测
中断机制: 用户可随时发送新消息中断当前处理。系统会安全终止正在进行的工具调用和LLM推理，立即处理新消息。

🤖 Agent 核心系统

AIAgent 是系统的中枢，负责推理循环、工具调度和上下文管理

🔄
AIAgent 主循环
run_agent.py — 核心推理引擎
  • 维护迭代预算 (IterationBudget)，线程安全计数
  • 支持工具并行执行，提高效率
  • 上下文超限时自动压缩历史消息
  • 完整的错误分类与故障转移机制
  • 支持中断处理，用户可随时打断
🧩
PromptBuilder 提示构建
agent/prompt_builder.py
  • 构建系统提示：身份、能力、约束
  • 注入 MEMORY.md 和 USER.md 内容
  • 加载激活的技能指令
  • 添加平台特定的格式提示
  • 注入自我演进策略 (StrategyInjector)
📦
ContextCompressor 上下文压缩
agent/context_compressor.py
  • 保护重要消息（首尾轮次）不被压缩
  • 使用 LLM 摘要化中间历史消息
  • 基于 token 计数精确控制压缩比
  • 支持模型特定的上下文窗口适配
🧠
MemoryManager 记忆管理
agent/memory_manager.py
  • 持久化记忆 (MEMORY.md) — 跨会话保持
  • 用户画像 (USER.md) — 用户偏好与信息
  • 记忆清洗与格式标准化
  • 定期记忆提醒（nudging 机制）
⚡
ErrorClassifier 错误分类
agent/error_classifier.py
  • 分类 API 错误：速率限制/认证/超时/内容过滤
  • 决定故障转移策略：重试/切换模型/降级
  • 指数退避 + 抖动重试机制
📂
SubdirectoryHintTracker
agent/subdirectory_hints.py
  • 跟踪当前工作目录上下文
  • 为文件操作提供路径提示
  • 支持多级目录导航

🎭 多智能体编排

支持多 Agent 协作，提供并行、串行、MapReduce、DAG 四种编排模式

🔄
parallel 并行模式
多 Agent 同时执行

多个 Agent 同时处理不同子任务，结果汇总后返回。适用于独立子任务并行处理。

➡️
sequential 串行模式
按序依次执行

Agent 按顺序依次执行，前一个的输出作为后一个的输入。适用于有依赖关系的流水线任务。

🗺
map_reduce 模式
分而治之

将任务拆分给多个 Agent 并行处理(map)，再由汇总 Agent 合并结果(reduce)。适用于大规模数据处理。

🔗
DAG 有向无环图
复杂依赖编排

定义任务间的复杂依赖关系，自动拓扑排序执行。支持条件分支和并行执行路径。

📖 使用指南

快速上手 Hermes Agent 的各项功能

快速开始

1

安装 Hermes

通过一键脚本安装：bash setup-hermes.sh，或使用 Docker：docker-compose up -d

2

配置 API Key

在 .env 或 cli-config.yaml 中设置 OpenRouter API Key（或其他模型 Provider 的 Key）

3

启动 CLI

运行 python cli.py 进入交互式对话。使用 -q 参数进行单次查询。

4

选择工具集

通过 --toolsets web,terminal,file 选择需要的工具集，或使用默认全量工具。

5

接入平台

配置平台 Token 后启动网关模式：python cli.py --gateway，自动连接所有配置的平台。
